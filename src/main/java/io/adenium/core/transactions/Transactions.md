# Adenium Transactions
## Transaction (Free)
A transaction that requires a recipient address, amount, and signature.
An address qualifies for 24 free transactions every 7200 blocks.
## Transaction To Alias (Free)
A Transaction that requires a recipient alias, amount, and signature.
An address qualifies for 24 free transactions every 7200 blocks.
## Transaction
A transaction that requires a recipient address, amount, fee, and signature.
## Transaction To Alias
A transaction that requires a recipient alias, amount, fee, and signature.

If the alias is not yet registered, transaction will fail.
## Make (Token)
A transaction that takes token parameters and produces a token.
## Make (NFT)
A transaction that takes NFT parameters and produces an NFT.
